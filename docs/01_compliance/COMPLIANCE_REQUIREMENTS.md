## Contenus sensibles nécessitant validation avant publication

Les contenus ci-dessous doivent être relus et approuvés par Berlioz Conseil :

- toute mention d'indépendance du conseil ;
- toute référence à l'optimisation fiscale ;
- toute référence à un rendement ou une performance ;
- toute référence à une allocation d'actifs ;
- toute mention de protection du capital ;
- toute comparaison de produits ;
- toute mention de partenaires financiers ;
- toute mention de compagnies d'assurance ;
- toute référence à des certifications spécifiques ;
- toute mention du statut CIF ;
- toute mention du statut IOBSP ;
- toute mention du statut immobilier ou carte T ;
- toute mention relative au courtage en assurance ;
- toute mention de l'appartenance CNCGP ;
- toute référence client ou témoignage.

---

## Termes interdits sans validation préalable

Les formulations suivantes ne doivent jamais apparaître sans validation explicite :

- rendement garanti ;
- performance assurée ;
- optimisation fiscale garantie ;
- protection totale du capital ;
- meilleur placement ;
- solution sans risque ;
- indépendance totale ;
- accès privilégié ;
- exclusivité absolue ;
- conseil certifié (sauf certification précise et fournie) ;
- spécialiste reconnu (sauf justification vérifiable) ;
- petits porteurs ;
- clients non qualifiés ;
- demandes non sérieuses.
